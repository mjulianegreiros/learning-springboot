# Desafio Spring Boot - API de Itens

## Tasks

1. Criar a entidade Item com atributos básicos (id, nome, descrição).
2. Configurar o repositório utilizando Spring Data.
3. Implementar método POST para criar um Item.
4. Implementar método GET para retornar um Item por id.
5. Implementar método GET para retornar uma lista de Items.
6. Criar testes unitários para:
   - criação de Item
   - busca por id
   - listagem de Items
7. Garantir que os testes validem o comportamento esperado.
