# Spring Boot GET Exercise

## Objetivo
Implementar métodos GET em um controller usando Spring Boot.

## Tarefas
1. Criar um endpoint GET `/users` que retorna uma lista de usuários.
2. Criar um endpoint GET `/users/{id}` que retorna um usuário por ID.
3. Se o usuário não existir, retornar 404.

## Desafio extra
- Adicionar um endpoint `/users/search?name=...` que filtra por nome.
